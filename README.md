# lionbot_ts

This project is built with [Bun](https://bun.sh), a fast all-in-one JavaScript runtime and drop-in replacement for Node.js & npm.

To install dependencies:

```bash
bun install
```

To run the leaderboard script:

```bash
bun run leaderboards
```

To run the site:

```bash
bun run dev
```
